# Launcher Theme Enhancer

整合 **城市天气**、**动态视频背景**、**背景粒子特效** 三个功能为一体的 SJMCL 扩展。

提供两个版本：
- **英文版**（org.yoshino.launcher-theme-enhancer）
- **中文版**（org.yoshino.launcher-theme-enhancer.zh）

## 功能

### 城市天气
- 通过 Open-Meteo API 获取实时天气数据
- 覆盖全球 200+ 城市
- 天气卡片显示温度、湿度、风速、体感温度
- 天气匹配的粒子背景（雨、雪、雷暴等）
- 氛围色调叠加开关

### 动态视频背景
- 上传 MP4/WebM/OGG 视频文件
- 始终在最底层渲染
- 音量控制滑块
- 重启后保留已上传的视频
- 播放/暂停/删除控制

### 粒子特效（10 种）
- 雨、雪、沙暴、星空、极光
- 落叶、水下、火焰、日落、Minecraft 风格

## 层级系统
- 视频背景：最底层
- 城市天气粒子：中间层
- 主题粒子：最上层

## 安装方式
1. 从 Releases 下载对应的 .sjmclx 文件
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 选择文件并启用即可
