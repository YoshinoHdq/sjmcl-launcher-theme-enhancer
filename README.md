## 简介

Launcher Theme Enhancer 是一款 SJMCL 扩展，将此前三个独立插件整合为一个无缝体验，同时解决了三者同时开启时的冲突问题。

内置中英文切换按钮，可在设置面板中一键切换语言，无需安装不同语言版本。

不支持与 sjmcl-video-background、sjmcl-weather-effects、sjmcl-city-weather 等旧版独立插件共存，建议卸载旧插件后使用本整合版以获得最佳体验。

## 功能特色

### 🌤 城市天气
- 通过 Open-Meteo API 获取实时天气数据
- 覆盖全球 200+ 城市
- 相比旧版，天气响应更精准、过渡更平滑

### 🎬 动态视频背景
- 上传 MP4/WebM 视频文件作为启动器背景
- 修复了旧版动态背景会覆盖其他插件页面的问题
- 音量控制、播放/删除

### ✨ 粒子特效（15 种）

- 附魔 — 浮动的附魔符文在空中飘浮
- 幽匿 — 循声守卫风格的幽匿粒子效果
- 冻结 — 冰雪霜冻覆盖整个界面
- 代码 — 矩阵风格的绿色代码雨
- 灰尘 — 空气中漂浮的细微尘埃
- 活力 — 充满生机的彩色光点
- 火焰 — 跳动的火焰粒子效果
- 雪 — 缓缓飘落的雪花
- 日落 — 暖色调的日落余晖
- 水下 — 水中的气泡与光影
- 极光 — 流动的极光带效果
- 雨 — 淅淅沥沥的雨滴
- 沙暴 — 漫天飞舞的沙尘
- 星空 — 闪烁的星辰夜空
- 落叶 — 随风飘落的叶子

### 🌐 语言切换
- 设置面板中一键切换中英文
- 不再需要下载不同语言版本

> ⚠️ **注意：已移除粒子和动态视频的记忆功能**，每次打开启动器需重新设置。对于此改动深表歉意，由于技术力的原因，短时间内不考虑加回来。

## 使用方法

1. 在 Releases 下载 org.yoshino.launcher-theme-enhancer-1.0.0.sjmclx
2. 打开 SJMCL → 拖入启动器并启用
3. 在首页卡片展开操作即可

## 兼容性

- 最低启动器版本: 1.1.3
